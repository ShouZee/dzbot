# Telegram bot for homework collection

Бот на Node.js принимает домашние задания от учеников, просит выбрать класс при первом запуске, затем запрашивает имя и фамилию и пересылает работу преподавателю в Telegram.

## Что умеет бот

- принимает `document`, `photo` и `video`
- при первом запуске просит ученика выбрать класс кнопкой
- запоминает выбранный класс
- просит ученика ввести имя и фамилию только один раз
- дает ученику кнопку для смены класса
- дает ученику кнопку для смены имени и фамилии
- разрешает менять класс самостоятельно не чаще одного раза в 7 дней
- позволяет запросить досрочную смену класса через подтверждение преподавателя
- если имя и фамилия еще не сохранены, просит ввести их один раз
- пересылает файл в отдельный чат класса или в основной чат преподавателя
- позволяет преподавателю отправлять домашние задания в чат нужного класса через бота
- позволяет добавлять дополнительных учителей по Telegram ID
- позволяет удалять дополнительных учителей через кнопку
- добавляет подпись с ФИО ученика, классом, username и Telegram ID

## Управление преподавателя

- `Добавить класс` - бот попросит ввести название нового класса
- `Показать классы` - бот покажет текущий список
- `Удалить класс` - бот покажет кнопки с классами для удаления
- `Переименовать класс` - бот покажет кнопки с классами, затем попросит новое название
- `Назначить чат классу` - бот попросит выбрать класс и отправить `chat ID` нужного чата
- `Показать чаты классов` - бот покажет, в какой чат отправляется каждый класс
- `Задать домашнее задание` - бот попросит выбрать класс, затем можно отправить текст, документ, фото или видео в чат этого класса
- `Добавить учителя` - бот попросит Telegram ID нового учителя и даст ему доступ к меню преподавателя
- `Удалить учителя` - бот покажет список дополнительных учителей и удалит выбранного

## Как работает сценарий

1. Преподаватель добавляет классы через кнопки.
2. Ученик открывает бота и нажимает `/start`.
3. Бот просит выбрать класс кнопкой.
4. Ученик один раз выбирает свой класс.
5. Ученик отправляет файл.
6. Бот просит написать имя и фамилию.
7. Бот пересылает работу в чат, назначенный для этого класса.

## Где хранятся данные

- `data/classes.json` - список классов
- `data/students.json` - сохраненные классы учеников
- `data/class-chats.json` - привязка классов к чатам

## Запуск

```bash
npm start
```

## Файлы проекта

- `index.js` - основной код бота
- `.env` - токен бота и Telegram ID преподавателя
- `data/classes.json` - классы
- `data/students.json` - выбранные учениками классы
- `data/class-chats.json` - чаты для классов
